Sobre mí

Ingeniero en Ciencias de la Computación egresado de UNICAH, con experiencia práctica en desarrollo de software y administración de infraestructura. He trabajado con tecnologías como C#, JavaScript, ASP.NET, SQL Server y MySQL en proyectos orientados a resolver problemas concretos de negocio.


Durante mi práctica profesional en la Secretaría de Finanzas (SEFIN), lideré el desarrollo de Mi SEFIN Móvil, una aplicación móvil institucional ciudadana construida con Ionic, Angular y Capacitor (HTML, CSS, TypeScript), con integración a SharePoint vía Power Automate. Además, desarrollé una aplicación de gestión de agenda integrada con Microsoft Outlook usando Power Apps, ambas desplegadas en entorno institucional real. Me caracterizo por aprender rápido, adaptarme a nuevos entornos tecnológicos y entregar soluciones funcionales con impacto real.

Proyectos

01

Mi SEFIN Móvil Destacado

Aplicación móvil institucional ciudadana desarrollada durante la práctica profesional en la DGTI de la Secretaría de Finanzas (SEFIN). Permite a los ciudadanos hacer seguimiento de expedientes, ver consultas de pagos, consultar ubicaciones e información institucional. Construida con Ionic, Angular y Capacitor (HTML, CSS, TypeScript), con integración a SharePoint mediante flujos de Power Automate. Módulos de OTP, formularios de quejas y confirmación visual con identidad de marca institucional.

Ionic Angular Capacitor TypeScript HTML / CSS Power Automate SharePoint

Power Apps Power Fx Office 365 Outlook API Microsoft Exchange
